# Weather App

## Description
Weather App is a responsive web application that allows users to check real-time weather information for selected cities. The app fetches data from a public API and displays details such as temperature, feels-like temperature, humidity, wind speed, wind gust, and overall weather conditions. It features a clean, user-friendly interface for easy weather tracking.

## Features
- Select a city from a dropdown menu.
- Fetch current weather information via API.
- Display weather icon, main condition, temperature, feels-like temperature, humidity, wind speed, and wind gust.
- Responsive design for different screen sizes.
- Handles API errors gracefully.

## Technologies Used
- **HTML** – structure of the application.
- **CSS** – styling and layout.
- **JavaScript** – fetching and displaying weather data, DOM manipulation.
- **Fetch API** – retrieving data from a weather API.

## Usage
1. Open `index.html` in a browser.
2. Select a city from the dropdown menu.
3. Click **Get Weather** to view the current weather information.
